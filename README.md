# 📱 مولّد أوامر ADB

أداة ويب تساعدك على **توليد أوامر ADB** لتغيير لغة نظام أندرويد أو لغة تطبيق معين، مع دعم نمط الأرقام واتجاه الواجهة — دون الحاجة لحفظ أي أوامر.

## 🌐 جرّب الأداة الآن

**[← اضغط هنا لتجربة مولّد أوامر ADB](https://iofahmawi.github.io/adb/)**

## ✨ المميزات

- 🌍 توليد أمر تغيير لغة النظام مع اختيار الدولة واللغة ونمط الأرقام
- 📦 توليد أمر تغيير لغة تطبيق محدد عبر اسم الحزمة (Package Name)
- 🔢 دعم أنماط الأرقام: غربية (1234) أو مشرقية (١٢٣٤)
- 🔁 زر تبديل بين وضع `adb shell` والوضع المباشر (داخل الجهاز)
- 📋 نسخ الأمر بضغطة واحدة
- 📲 يدعم تثبيته كتطبيق على الجهاز (PWA)
- 📱 متجاوب مع جميع أحجام الشاشات

## 🛠️ التقنيات المستخدمة

- HTML5 + CSS3 + JavaScript (Vanilla JS)
- Service Worker لدعم PWA

## 📊 وظائف الأداة

| التبويب | الوظيفة |
|---|---|
| لغة النظام | يولّد أمر `settings put system system_locales` مع اللغة والدولة ونمط الأرقام |
| لغة التطبيق | يولّد أمر `cmd locale set-app-locales` لإجبار تطبيق على اللغة الإنجليزية (يتطلب أندرويد 13+) |
