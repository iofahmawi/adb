## 📱 مولّد أوامر ADB

أداة ويب تساعدك على توليد أوامر ADB لتغيير لغة نظام أندرويد أو لغة تطبيق معين، مع دعم نمط الأرقام واتجاه الواجهة — دون الحاجة لحفظ أي أوامر.

### 🔗 تجربة الأداة
* [🚀 معاينة حية للأداة](https://iofahmawi.github.io/adb/)

### ✨ المميزات

* **توليد أمر لغة النظام**: اختيار الدولة واللغة ونمط الأرقام بسهولة.
* **توليد أمر لغة التطبيق**: تغيير لغة تطبيق محدد عبر اسم الحزمة (Package Name).
* **دعم أنماط الأرقام**: غربية (1234) أو مشرقية (١٢٣٤).
* **زر التبديل**: للتبديل بين وضع `adb shell` والوضع المباشر داخل الجهاز.
* **نسخ سريع**: نسخ الأمر الناتج بضغطة واحدة.
* **تطبيق ويب تقدمي (PWA)**: يدعم التثبيت على الجهاز والاستخدام بدون متصفح.
* **واجهة متجاوبة**: تعمل بكفاءة على جميع أحجام الشاشات.

### 🛠 التقنيات المستخدمة

* **HTML5 & CSS3 & JavaScript (Vanilla)**: بدون أي أطر عمل خارجية.
* **Service Worker**: لدعم PWA.

### 📊 وظائف الأداة

| التبويب | الوظيفة |
|--------|---------|
| لغة النظام | يولّد أمر `settings put system system_locales` مع اللغة والدولة ونمط الأرقام |
| لغة التطبيق | يولّد أمر `cmd locale set-app-locales` لإجبار تطبيق على اللغة الإنجليزية (يتطلب أندرويد 13+) |
